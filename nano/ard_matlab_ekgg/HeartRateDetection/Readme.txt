1) Download the ILI9341 and Adafruit GFX library - 

https://github.com/adafruit/Adafruit_ILI9341 and https://github.com/adafruit/Adafruit-GFX-Library 

2) Place them inside the tftShield/include folder here. 

3) Add the current folder and all subfolders to your MATLAB path. 

4) Open HeartSignals.slx from inside the SimulinkModels folder and press deploy to hardware button.
